# Get the Edge Table from a ComplexMap Object

A simple accessor function to extract the tibble of network edges from a
\`ComplexMap\` object.

## Usage

``` r
getEdgeTable(cm)
```

## Arguments

- cm:

  A \`ComplexMap\` object, typically the output of
  \`createComplexMap()\`.

## Value

A \`tibble\` containing the edge data.
